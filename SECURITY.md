# NumbrelOs Security

NumbrelOs bir prototip olduğundan aşağıdaki güvenlik önlemlerini dikkate alın:

- Üretim ortamına geçmeden önce kimlik doğrulama ekleyin.
- Hassas servisleri internete açmayın.
- Konteyner ve sistem güncellemelerini düzenli olarak yapın.
- Gerçek verilerle test etmeden önce yedekleme stratejisi oluşturun.

Bu dosya, NumbrelOs için temel güvenlik rehberi olarak eklenmiştir.
